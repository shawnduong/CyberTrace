CyberTrace Administrative Panel

Welcome to the CyberTrace administrative panel. Here, you can manage accounts. Accounts of type 0 are administrative accounts, type 1 are user accounts, and type 2 are pending accounts. For security reasons, pending accounts require administrator approval before they can become user or administrative accounts. All new registrations are pending by default.

Documentation

CyberTrace's knowledge base is currently a work in progress and contains limited information about the project as of now, but it is continually being built and authored over time.

Documentation about CyberTrace can be found on CyberTrace's official wiki.

Contributing

CyberTrace is a work in progress. Contributors are welcome to fork, commit, and send PRs, and PRs that are useful to the project will be approved. Alternatively, you can contribute by finding and reporting bugs, opening or responding to issues, and/or writing documentation.

More information about contributing can be found on CyberTrace's Contribution Guide.
